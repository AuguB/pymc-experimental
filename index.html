PyMC experimental¶

Where we grow the next batch of cool PyMC features

Dependencies¶

PyMC experimental is tested on Python 3.8+ and depends on PyMC V4.

Installation¶

pip install git+https://github.com/pymc-devs/pymc_experimental.git
